Algora : it's a program that you can created a wordlist of numbers
between two given number


run program :


	On windows :

		-pip install termcolor
		-pip install pyfiglet
		-pip installdatetime  (maybe instaled before)
		-on command prompt type >> 'py algora.py' or 'python algora.py

	On linux :

		-sudo apt-get install -y python-termcolor
		-sudo apt-get install -y python-pyfiglet
		-sudo apt-get install -y python-datetime    (maybe installed)
		-type on terminal : 'python3 algora.py'
